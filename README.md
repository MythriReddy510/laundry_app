# Laundry Order Management System

##  Setup Instructions

1. Install dependencies:
   pip install flask

2. Run the app:
   python app.py

3. Open:
   http://127.0.0.1:5000/

---

##  Features Implemented

- Create Order (with billing)
- Update Order Status
- View Orders
- Filter Orders (by status & phone)
- Dashboard (total orders, revenue, status count)
